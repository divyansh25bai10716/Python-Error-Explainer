# Python Error Explainer

## Project Overview

This project is an offline Python Error Explanation System that explains Python errors using a dataset and loop-based matching.

## Problem Statement

Beginner programmers often struggle to understand Python error messages. This project helps users understand errors in simple language.

## Features

- Offline project
- Dataset-based error explanations
- Loop-based matching
- Easy to expand dataset

## Technologies Used

- Python
- Pandas
- CSV Dataset

## How It Works

1. User enters Python error
2. Program searches dataset
3. If match found → explanation shown
4. If no match → new error message shown
